This file accompanies the paper:

Appraisal Theories for Emotion Classification in Text

Authors:
* Jan Hofmann (jan.hofmann@ims.uni-stuttgart.de)
* Enrica Troiano (enrica.troiano@ims.uni-stuttgart.de)
* Kai Sassenberg (k.sassenberg@iwm-tuebingen.de)
* Roman Klinger (roman.klinger@ims.uni-stuttgart.de)

For questions, please contact roman.klinger@ims.uni-stuttgart.de

In the tab separated file in this zip archive, you find columns for each cognitive appraisal dimension. The number in the respective column corresponds to the number of annotators who decided the respective dimension holds.
